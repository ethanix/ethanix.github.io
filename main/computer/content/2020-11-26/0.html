How to deploy RSS on your vps(centos7)

Antoine Lee

Prerequisites

  curl

  firewall

1. Set up environment for docker

Do the following things in the shell: 


    wget get.docker.com/gpg
    romkeys --import ./gpg

    yum provides '*/applydeltarpm'
    yum install deltarpm -y

    yum install docker-ce -y

2. Configure firewall for RSS

RSS needs an open port to support service successfully 


    firewall-cmd --zone=public --add-port=9000/tcp --permanent
    firewall-cmd --zone=public --list-ports

3. Install docker

After that, enter the commands below in the shell to download and install docker 


    curl https://get.docker.com | bash

    docker run -d --name ttrssdb nornagon/postgres
    docker run -d --link ttrssdb:db -p 9000:80 -e SELF_URL_PATH=http://121.196.195.91:9000 fischerman/docker-ttrss

See ports and services 


    netstat -ntlp
